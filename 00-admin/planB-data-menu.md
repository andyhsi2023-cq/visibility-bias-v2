## 2. 关键发现(标准品)
1. **年鉴物理存量 = 论文新行为骨架**:2002-2024、297 城、缺失极低。可让 VAI 从"文本"升级为"文本 × 真实存量"。
2. **CLASS B24 复活被否的微观基础**:审稿人 C4 说 CFPS 替代题不测机制;CLASS 有真·分设施满意度(可见 vs 功能),能正面做。代价:仅老年。
3. **夜间灯光 = 现成的"可见显著度"客观代理**,1992-2022 全覆盖。
4. **「示范」集中在计划段(75%)**:与你的"关注计划段"洞见有张力——计划段是 FP 重灾区;但**词典删「示范」后**,计划段恰是最干净的"战略意图"度量。两者需配套。
5. **官员(κ)+ 财政(B)调节齐备**,可检验模型比较静态(偏差随 κ、B 上升)。

## 3. 推荐 Plan-B 设计(新论文身份)
**从"一个文本测量工具"→"一个被真实行为与后果三角验证的可见性偏差度量体系"**:

- **测量层(修)**:精修可见性词典(删「示范」等多义词)+ **plan 段 VAI** + **位置加权**(你三点洞见),用人工+集成锚定验证。
- **行为效度(新,杀手锏)**:text-VAI ↔ **年鉴物理存量可见占比**(2002-2024)——直接证明"话术追踪真实分配",远强于原来"modest"的 CIR(r=0.24)。⚠️**先小验证这个相关有多强**。
- **机制(新)**:夜间灯光/绿地遥感作可见显著度;官员(κ)、财政(B)调节,检验模型预测。
- **后果(新,替代福利计算)**:高可见偏差城市 → 更差功能后果(污水处理率/空气优良天数)+ CLASS 老年居民对功能设施更不满。
- **第三方文本效度(修 H4)**:CNKI/人民日报政策文本替换失败的 Wikipedia。

## 4. 诚实边界 / G-R 裁决
- **招投标**:浙江 + 近两年为主 → 只能做**省级试点/概念验证**,非全国面板。
- **CLASS**:仅老年样本 → 微观后果带"老年"限定词。
- **官员 2018 截断 / VAI 到 2024**:调节分析窗口对齐到 ≤2018,或接受右截断。
- **媒体、内涝/事故 = 真空白**:本地没有,要采集(成本高),建议**不纳入本轮**。
- **仍无干净因果识别**:官员更替当"冲击"内生且 2018 截断——只能做**异质性/调节**(加固描述与机制),别再宣称因果(承接上轮 Plan-B 因果可行性=否的结论)。
- **构造注意**:年鉴指标是%/人均/密度混合单位——**必须各自年内标准化再合成**可见/功能综合分,不能直接相除(侦察兵的"均值0.362"是朴素相除,仅占位)。
